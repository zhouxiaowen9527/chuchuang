## 🚀 快速启动

### 方法一：双击批处理脚本

1. **双击 `start.bat`**
   - 自动安装依赖
   - 自动检查配置
   - 启动服务

2. **编辑 `.env` 文件**
   - 填入 Neon 数据库连接字符串

3. **双击 `init-db.bat`**
   - 创建默认账号和示例数据

### 方法二：命令行操作

```bash
# 1. 进入项目目录
cd C:\Users\Administrator\.qclaw\workspace\blueberry-payroll

# 2. 安装依赖
npm install --registry=https://registry.npmmirror.com

# 3. 配置环境变量
copy .env.example .env
# 编辑 .env，填入 DATABASE_URL

# 4. 在 Neon 控制台执行 database/schema.sql

# 5. 初始化数据
npm run init-db

# 6. 启动服务
npm run dev
```

## 🎯 核心功能

### 1. 工作记录管理
- 按小组录入工作记录
- 自动计算小组总收入
- 自动计算人均收入
- 为每个成员创建记录

### 2. 工资单管理
- 月度自动生成工资单
- 支持奖金、扣除调整
- 支持批量确认、发放
- 完整状态流转（draft → confirmed → paid）

### 3. 权限控制
- **员工**：查看个人工作记录和工资单
- **部门经理**：管理本部门所有数据
- **管理员**：全局管理权限

### 4. 审计日志
- 记录所有关键操作
- 支持按用户、操作类型筛选
- 支持导出为 CSV

### 5. 数据导出
- 工作记录导出为 CSV
- 工资单导出为 CSV
- 审计日志导出为 CSV
- 全量数据备份为 JSON
