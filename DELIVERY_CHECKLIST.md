## 🔑 核心功能

### 1. 工作记录录入（RecordService）
```
输入：小组、日期、多条计件规格
处理：
  1. 获取小组成员
  2. 计算小组总收入 = Σ(数量 × 单价)
  3. 计算人均收入 = 小组总收入 / 成员数
  4. 为每个成员创建一条记录
输出：工作记录列表
```

### 2. 工资单生成（PayrollService）
```
输入：周期（开始日期、结束日期）
处理：
  1. 查询周期内有工作记录的员工
  2. 汇总每个员工的计件收入
  3. 创建/更新工资单
  4. 支持奖金、扣除调整
  5. 支持批量确认、发放
输出：工资单列表
```

### 3. 权限控制（RBAC）
```
角色：
  - employee（员工）：查看个人数据
  - dept_manager（部门经理）：管理本部门
  - admin（管理员）：管理全局

权限矩阵：
  - 员工：查看个人工作记录、工资单、修改个人信息
  - 部门经理：管理本部门的小组、员工、计件规格、工作记录、工资单
  - 管理员：管理全局所有数据、审计日志、数据维护
```

```bash
POST /api/auth/login
{
  "email": "admin@blueberry.com",
  "password": "admin123"
}

Response:
{
  "code": 0,
  "message": "登录成功",
  "data": {
    "token": "eyJhbGc...",

  }
```

### 录入工作记录
```bash
POST /api/records
Authorization: Bearer <token>
{
  "group_id": "uuid",
  "work_date": "2026-03-27",
  "items": [
    { "spec_id": "uuid", "quantity": 100 },

Response:

  "data": {
    "group_total": 1000,
    "member_count": 4,
    "per_person": 250,
    "records_count": 8
  }
}
```
